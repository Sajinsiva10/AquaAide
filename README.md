# AquaAide
Conception et développement d’une application web interactive visant à faciliter l’accès à l’eau potable et aux maraudes solidaires en Île-de-France pour les personnes en situation de précarité.

L’application repose sur une carte interactive Leaflet intégrant :

- la géolocalisation de l’utilisateur
- le calcul des distances en temps réel
- le filtrage dynamique des services (points d’eau / maraudes)
- l’affichage de données détaillées (horaires, accessibilité, mises à jour)
- l’intégration de formulaires collaboratifs pour signaler ou ajouter des points

Le projet inclut également une logique complète de gestion des données côté front-end, une navigation multi-vues (carte, liste, statistiques, détails), ainsi qu’une attention particulière portée à l’UX/UI et à l’impact social.

Compétences : HTML, Feuilles de style en cascade (CSS), +3 compétences
